Rename nightwatch → noxaudit and prepare for open source release

## Background

Nightwatch is an AI-powered codebase auditor that runs nightly with rotating focus areas (security, patterns, docs, etc.). It's currently branded as "nightwatch" but that name is extremely common on GitHub (~1M repos). Renaming to "noxaudit" (nox = night in Latin) for uniqueness and potential open source release.

## Tasks

### 1. Repository-wide Rename

**Find and replace across all files:**
- `nightwatch` → `noxaudit` (case-sensitive)
- `Nightwatch` → `Noxaudit` (in titles, docs)
- Keep Python package structure: `nightwatch/` directory → `noxaudit/`

**Files to update:**
- All `*.py` files (imports, docstrings, comments)
- All `*.md` files (README, docs)
- `pyproject.toml` (package name, URLs)
- `action/action.yml` (GitHub Action)
- Test files
- Config examples

**Special cases:**
- `.nightwatch/` directory references in code → `.noxaudit/`
- `nightwatch.yml` config file → `noxaudit.yml`
- `nightwatch-ai` PyPI package name → `noxaudit` (simpler, more distinctive)
