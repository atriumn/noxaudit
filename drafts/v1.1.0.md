Noxaudit v1.1.0 is now available.

This release builds on the stability fixes from v1.0.1, focusing on hardening the core audit engine and streamlining our distribution pipeline for better reliability.

### What’s New

* **Automated Release Workflows**: We’ve integrated automated versioning and changelog generation to ensure a more predictable and transparent update cycle.
* **Performance Optimization**: Improved handling of high-concurrency scan requests to reduce memory overhead.
* **Dependency Hardening**: Updated internal dependencies to resolve potential vulnerabilities and improve runtime stability.

### Why it matters

Reliability is non-negotiable for audit tooling. By automating the "boring" parts of our release process, we’ve cleared the path for faster iteration on our scanning logic while ensuring that critical patches reach you immediately.

### Get Started

Review the full changelog and download the latest assets on GitHub:
[Noxaudit v1.1.0 Release](https://github.com/atriumn/noxaudit/releases/tag/v1.1.0)
