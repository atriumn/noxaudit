Noxaudit has officially hit its 1.0 milestone. Following the release of v1.0.0, version **1.0.1** focuses on documenting our performance and ensuring model accuracy for production environments.

### What's New

*   **Benchmark Results:** We have published comprehensive benchmark data. You can now audit our performance metrics directly in the documentation to see how Noxaudit handles various workloads.
*   **Model Reference Updates:** We've updated our internal model references to ensure full compatibility with the latest standards.
*   **Core Stability:** With the 1.0 release branch now active, our API is stabilized for production use. This update clears the path for predictable integrations.

### Why it Matters

The move to 1.0.1 isn't just a version bump; it's about transparency. By publishing our benchmarks, we're giving you the data needed to validate Noxaudit's audit precision before you deploy.
