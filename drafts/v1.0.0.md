Noxaudit v1.0.0 is now available. This release centers on standardizing our benchmarking pipeline to provide more rigorous audit metrics.

### What’s New

*   **Automated Benchmark Runner:** We've introduced a dedicated runner script and scorecard analysis tool. You can now execute structured audits and generate performance reports programmatically rather than relying on manual intervention.
*   **Claude 3.5 Haiku & Matrix Expansion:** Support for `claude-haiku-4-5` has been added to our pricing logic, and the `corpus.yml` model matrix has been expanded. This allows for broader, more cost-aware comparisons across the latest provider endpoints.

### Impact

These changes transform Noxaudit from a collection of utilities into a cohesive testing framework. By leveraging the new scorecard analysis and expanded model matrix, teams can now quantify model reliability and cost-efficiency at scale.
